A light-weight library for implementing and trying out different diffusion based models for image generation and Timeseries Forecasting.

### Requirements
- Python >= 3.10
- PyTorch >= 2.3

### Implemented
- [x] VAE
- [x] UNet
- [x] ResNet
- [x] PixelCNN
- [x] PixelSNAIL
- [x] Encoder based Transformer
- [x] DDPM
- [ ] DDIM
- [ ] TimeMixer
- [ ] TimesNet
